# Color

## Default Color

Basically Andromeda Design System supports two `dynamic colors` that are neutral and primary color.

_<b>Dynamic colors</b>: They are used in css variable...meaning you can change their value by adding color scheme in theme._

The default color of all components will be neutral. Add prop `color="colorName"` to change color.

```tsx
import * as React from "react";
import {
  AndromedaDesignSystemProvider,
  Button,
} from "@andromedaprotocol/react";

function App() {
  return (
    <AndromedaDesignSystemProvider>
      <Button>Neutral</Button> // neutral color
      <Button color="primary">Primary</Button> // primary color
    </AndromedaDesignSystemProvider>
  );
}
```

## Adding Color

To add other colors edit the `colors` option in the @andromedaprotocol/core plugin of tailwindcss.

```js
// tailwind.config.js
import colors from "tailwindcss/colors";

module.exports = {
  theme: {
    extend: {
      colors: {
        pink: colors.pink,
        // name your color
        warning: colors.yellow,
      },
    },
  },
  // add Andromeda Design System plugin
  plugins: [
    require("@tailwindcss/forms"),
    require("@andromedaprotocol/core")({
      colors: ["pink", "warning"],
    }),
    // ...
  ],
};
```

```tsx
import * as React from "react";
import {
  AndromedaDesignSystemProvider,
  Button,
} from "@andromedaprotocol/react";

function App() {
  return (
    <AndromedaDesignSystemProvider>
      <Button color="warning">Primary</Button> // warning color
      <Button color="pink">Primary</Button> // pink color
    </AndromedaDesignSystemProvider>
  );
}
```
