Props

Switch
  • color[string]·Default: primary

  • Set the switch color

  • size[sm, md, lg, xl]·Default: md

  • Size of the switch

  • disabled[boolean]·Default: false

  • Makes switch disabled

  • invalid[boolean]·Default: false

  • Makes switch invalid

  • defaultChecked[boolean]·Default: false

  • The switch will be initially checked

  • checked[boolean]·Default: false

  • The switch will be checked

  • id[string]·Default: ''

  • Switch id

  • name[string]·Default: ''

  • Switch name

  • value[string, number]·Default: ''

  • Switch value

  • onChange[React.ChangeEventHandler<HTMLInputElement>]·Default: ''

  • The callback invoked when the checked state of the `Switch` changes..

  • aria-label[string]·Default: ''

  • A label that describes the input

  • aria-describedby[string]·Default: ''

  • The id of the element that describes the input.

  • aria-labelledby[string]·Default: ''

  • Refers to the id of the element that labels the checkbox element.
