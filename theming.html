Theming

By default all Andromeda Design System components use the default theme. You can extend and customize theme on top level with AndromedaDesignSystemProvider.

AndromedaDesignSystemProvider

AndromedaDesignSystemProvider component allows you to change theme globally. It is not required if you decide to stick with default theme:

import * as React from "react";​// 1. import `AndromedaDesignSystemProvider` componentimport {  AndromedaDesignSystemProvider,  Button,} from "@andromedaprotocol/react";​function App() {  return <Button>Button</Button>;}​function ThemedApp() {  // 2. Use at the root of your app  return (    <AndromedaDesignSystemProvider>      <App />    </AndromedaDesignSystemProvider>  );}

Theme structure

interface ColorScheme {  id: string;  type: "light" | "dark";  colors: {    bg: {      base: string; // used for background      fill: string; // used for navbar, sidebar, header    };    text: {      foreground: string; // text primary      muted: string; // text secondary    };    primary: Colors;    neutral: Colors;  };}​interface AndromedaTheme {  cursor: "default" | "pointer";  rounded: "0.375rem";  colorSchemes: Record<string, ColorScheme>;}

For a complete reference of available theme properties and their default values, see the default theme configuration.

Customize Theme

// 1. Import `extendTheme`import { extendTheme, colors } from "@andromedaprotocol/react";​// 2.Define new color schemeconst cool: ColorScheme = {  id: "cool",  type: "dark",  colors: {    bg: {      base: colors.coolGray["900"],      fill: colors.coolGray["900"],    },    text: {      foreground: colors.coolGray["100"],      muted: colors.coolGray["300"],    },    primary: colors.cyan,    neutral: colors.coolGray,  },};​// or custom default color schemeconst light: ColorScheme = {  id: "light",  type: "light",  colors: {    bg: {      base: colors.gray["800"],      fill: colors.gray["900"],    },    text: {      foreground: colors.gray["100"],      muted: colors.gray["300"],    },    primary: colors.purple,    neutral: colors.gray,  },};​// 3. Call `extendTheme` and pass your custom valuesconst theme = extendTheme({  cursor: "pointer",  colorSchemes: {    light,    cool,  },});​// 4. Pass the new theme to `AndromedaDesignSystemProvider`<AndromedaDesignSystemProvider theme={theme} colorScheme="cool">  <App /></AndromedaDesignSystemProvider>;// 5. Now you can use these colors in your componentsfunction Usage() {  return <Button color="primary">Welcome</Button>;}

Density mode

Andromeda Design System provide two density mode to apply to size of component: compact and comfortable

The following components have applyed density:

  • Button
  • Input
  • Select
  • Switch
<AndromedaDesignSystemProvider density="compact">  <App /></AndromedaDesignSystemProvider>

TypeScript

@andromedaprotocol/react package exports types:

  • AndromedaTheme – theme object interface
  • AndromedaThemeOverride – theme override options
  • ColorScheme - color scheme object interface
