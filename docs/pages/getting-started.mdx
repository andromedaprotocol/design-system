# Getting Started

## Installation

Andromeda Design System works out of the box with create-react-app (including TypeScript version), Preact cli (with compat), Next.js, Gatsby and any other environment.

To install the Andromeda Design System run the following within your project directory.

```bash
npm i @andromedaprotocol/core @andromedaprotocol/react @tailwindcss/forms

# OR

yarn add @andromedaprotocol/core @andromedaprotocol/react @tailwindcss/forms
```

## Configuration

The default theme of this library depends on the `@tailwindcss/forms` plugin. To use it, follow the steps on the plugin source page. https://github.com/tailwindlabs/tailwindcss-forms

Andromeda Design System can be configed in `tailwind.config.js`:

```js
// tailwind.config.js
module.exports = {
  mode: "jit",
  purge: [
    // ...
    "./node_modules/@andromedaprotocol/**/*.{js,ts,jsx,tsx}",
  ],
  darkMode: "class", // or 'media' or 'class'
  theme: {
    extend: {},
  },
  variants: {
    extend: {},
  },
  plugins: [require("@tailwindcss/forms"), require("@andromedaprotocol/core")],
};
```

**Note**: Andromeda Dessign System required `jit` mode and `tailwindcss` version `2.1.4` or higher (`basic` mode seems buggy when compiling some components)

## Quick start

Here's a quick example to get you started, **it's literally all you need:**

```jsx
import * as React from "react";
import {
  AndromedaDesignSystemProvider,
  Button,
} from "@andromedaprotocol/react";

function App() {
  return (
    <AndromedaDesignSystemProvider>
      <Button>Hello Andromeda Design System</Button>
    </AndromedaDesignSystemProvider>
  );
}
```
